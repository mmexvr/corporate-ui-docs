We've got your back

Can't find what you needed in our documentation section? Get in touch with a friendly member of our team.

Please send us an e-mail at cs.appsupport@scania.com and we'll be happy to help out.

We will get back to you as soon as we can.

Note: Scope of our support is Limited.

We can help you with many tasks, such as resolving the clarifications on the documentation provided, but some questions are out of scope. Like helping you write or debug code or consulting requests.

Here's a summary of the types of tasks we can help with, and also the ones we can't.

  • Questions about incorporating Scania Corporate UI Guidelines within the app. We can give you advice, but not detailed code.
  • Questions about a method or feature isn't working as documented/expected. We can help if you have a reliable example that shows the problem.
  • We cannot help with the technical questions about Architecting the Mobile Applications and debugging the code.
  • There is no support for IOS (APP or Screens) right now. Get started following Android UI and help will be provided for the UI part.

We provide full support in English. Non-English cases have limited support and exclude troubleshooting requests.
