Coding Standard for AEM

HTML & CSS for ComOn components

Creating components for use in the AEM publishing system differs a bit from creating stand alone applications. Basically, most of the things an editor needs is already in place and all that needs to be done is to develop for the current component. That is done using basic web technologies as HTML, CSS and JavaScript.

Corporate UI coding standard

ComOn components apply the same coding standard as other projects at Scanai and you'll use parts of Scania's Corporate UI framework (HTML, CSS classes and UI Components) when doing your implementation. An overview of the Corporate UI () can be found here:
