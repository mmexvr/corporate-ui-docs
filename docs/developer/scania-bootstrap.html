Scania Bootstrap

Scania Bootstrap is now a subset of the Corporate UI framework. Scania Bootstrap can still be used "stand alone" to tailor applications with the Corporate UI look and feel, but it will not fully take advantage of the cost cutting maintainance that the use of Web Components offer. You'll simply have to develop and maintain a lot of the HTML and JavaScript code already availble in the components.

When it comes to designing your particular functionallity, please grap code snippets from the the Design Patterns site. The library is uppdated on a daily basis so make sure you'll visit it frequently.

All things that you'll need for your development, like icons, fonts and CSS and Web Components Polyfills will become available for you once you're loading trhe corporate-ui.js.

Custom CSS

All the CSS that you'll need should be available using Boostrap Classes and Boostrap Components. From a maintainance perspective it's important that you do avoid creating custom CSS classes. If you have a need that is not covered in Bootstrap, please contact us and we'll find a solution for you.
