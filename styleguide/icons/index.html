Scania Icons

The design of the modern and custom-made icon system for communication is based on the Scania symbol, the experience throughout all applications. The icons enable clear and instantaneous communication for information graphics and sales support. The icons should always be supported by text (except for some UI icons used in web applications). Use the icons for information, never for decoration or as logotypes.

Restrictions: The icons may only be used for their intended purposes. Any restrictions must be followed. The icons may not be modified. Any questions may be submitted to Scania identity helpdesk via identity@scania.com.

Scania Sans font family and the product DNA. It has been customised with the goal of unifying the Scania brand.

Using the icons

The icons are available through CSS when using the Corporate UI framework. If you out want to use the icons without the Corporate UI framework, for example in a legacy application, you can include only the icons.css file and use the icons by the same CSS classes as described below (inspect each icon to obtain its CSS class).

Sizes and colors

The icons are currently available in a positive (Scania Blue) and negative (Scania White) color. By default you'll get them in Scania Blue and if you want to use the negative version, just add the class neg after the icon class. To show the icons in diffent sizes, use the classes xs, sm, md, lg and xl, xxl. For example:

<i class="scania-icon-email neg xl"></i>

Creating new icons

Even though we've aim at having an extensive icon library, chances are that you will not find the icon you need in our library. If you need to create your own icon, we've created these guidelines to help you get it right.

Icons for Scania applications

The Scania Icon Library tries to provide enough useful icons for your application, but in some cases you will need additional Icons. If you are unable to create that Icon according to the icon guidelines, we recommend you to use Font Awesome Pro Light icons to align with Scania Branding as much as possible. These can be used on web and be downloaded as SVG and used for mock-ups.
